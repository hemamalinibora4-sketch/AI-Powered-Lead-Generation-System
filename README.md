# AI-Powered Lead Generation System

Salesforce-based lead scoring and qualification system that combines native automation (Flow) with Agentforce (Einstein GPT) to score leads and generate AI-written summaries for sales representatives.

# Core Features

* Custom Lead fields: `Lead_Score__c`, `AI_Summary__c`, `Industry_Type__c`, `Engagement_Level__c`
* Validation rules enforcing required and well-formatted **Email** and **Company**
* Dedicated **Lead Generation App** (Lightning App) with Leads, Reports, and Dashboards
* Role hierarchy: **Sales Director → Sales Manager → Sales Representative**
* Record-Triggered Flow for automated lead scoring
* AI-generated lead summaries using **Agentforce (Einstein GPT)** and **Prompt Builder**
